# Service Template Standard

This document defines the baseline expectations for a backend service in OpsPilot AI.

## Required Files

Each backend service should eventually contain:

- `package.json`
- `tsconfig.json`
- `README.md`
- `.env.example`
- `service.manifest.json`

## Required Source Structure

```txt
src/
  domain/
  application/
  presentation/
  infrastructure/
  main.ts
```
